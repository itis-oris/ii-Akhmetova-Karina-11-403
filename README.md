# CakeShop

Интернет-витрина кондитерской сети с тремя точками в Казани.  
Пользователи могут просматривать кондитеров и их торты, оставлять отзывы, оформлять заказы.  
Администратор управляет тортами и кондитерами (CRUD).  
Интеграция с Яндекс.Картами: определение ближайшей кондитерской по адресу пользователя, расчёт расстояния, показ мастеров.

## Стек

- Java 17, Spring Boot 3.4.3
- Spring MVC + Freemarker (MPA) и REST API (`/api/**`) с OpenAPI / Swagger UI
- Spring Security — аутентификация по email, BCrypt, роли `USER` / `ADMIN`, CSRF (для форм, отключён для API)
- Spring Data JPA + Hibernate, кастомные JPQL (подзапрос для тортов дешевле средней цены), CriteriaBuilder (динамический поиск тортов)
- PostgreSQL
- Redis + Spring Cache (`@Cacheable` на геокодирование адресов)
- Docker + Docker Compose (PostgreSQL, Redis, приложение)
- Java HTTP Client (встроенный) — вызов Яндекс.Геокодера (внешнее API)
- Lombok, Bean Validation, AJAX

## Функциональность

- **Регистрация / вход** с BCrypt-паролями, роли `USER` и `ADMIN`.
- **Профиль пользователя**: указание телефона, адреса, просмотр своих заказов.
- **Геокодирование адреса** через Яндекс.API с кэшированием в Redis.  
  При сохранении адреса определяются координаты, на карте отображается **ближайшая точка сети**, расстояние, список кондитеров, работающих в ней, ссылка на Яндекс.Карты (AJAX).
- **Каталог кондитеров**: выбор мастера - просмотр его тортов, отзывы, оформление заказа.
- **Поиск и фильтрация тортов** в каталоге через AJAX (по названию, категории, артикулу) – динамическая подгрузка без перезагрузки страницы.
- **Отзывы** к тортам с рейтингом 1–5 (только для аутентифицированных).
- **Административная панель** (доступна только для `ROLE_ADMIN`):
  - CRUD тортов (название, артикул, категория, вес, цена, привязка к кондитеру)
  - CRUD кондитеров (имя, телефон, точка сети) – удаление запрещено, если есть заказы.
- **REST API** с полным CRUD для тортов (`/api/cakes`), GET для публичного просмотра, POST/PUT/DELETE – только для аутентифицированных.  
  Документация – `/swagger-ui.html`.
- **Нестандартный SQL**: JPQL-запрос с подзапросом для получения тортов дешевле средней цены по каталогу.
- **CriteriaBuilder**: динамический поиск тортов по фрагменту текста (название/категория/артикул) и минимальной цене.
- **Защита от XSS**: Freemarker автоматически экранирует HTML-символы.
- **Docker**: приложение, PostgreSQL и Redis поднимаются одной командой `docker-compose up -d`.
- **Тесты**: интеграционный (контекст), репозитория (`@DataJpaTest` с H2), веб-слоя (`@WebMvcTest` с моками).
